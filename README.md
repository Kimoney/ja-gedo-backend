# Ja Gedo Safaris Backend

## Built with Flask

## How To Start

1. Clone the Repo
2. Create a virtual env with 
   ```bash
   python -m venv env
   ```
3. Activate the environment
   - On Linux/Mac:
     ```bash
     source env/bin/activate
     ```
   - On Windows:
     ```bash
     env\Scripts\activate
     ```
4. Start the Server with 
   ```bash
   python run.py
   ```
5. Enjoy!!
